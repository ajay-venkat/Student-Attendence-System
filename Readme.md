# Smart Attendance System – Setup Guide

## Requirements

* Python 3.11
* Webcam

Install libraries:

pip install -r requirements.txt

---

## Project Files Needed

* attendance_system.py
* attendence_excel.xls
* Student 1.png 
* Student 2.png
* requirements.txt

Make sure the Excel file is **closed** before running.

---

## Step‑by‑Step Procedure

### 1. Open Project Folder

Open terminal inside the project directory.

### 2. Create Virtual Environment (optional but recommended)

python -m venv venv
venv\Scripts\activate

### 3. Install Dependencies

pip install -r requirements.txt

### 4. Run the Program

python attendance_system.py

### 5. Enter Lecture Name

Example:

AI Lab

### 6. Camera Starts

* Face will be detected automatically.
* Attendance will be saved in Excel.

### 7. Exit

Press **Q** to close the camera window.

---

## Output

Attendance will be stored in:

attendence_excel.xls
